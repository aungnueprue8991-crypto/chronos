# 🏺 CHRONOS: The Sovereign Restoration Engine

**Waking the Ancestors with 2026 AI.**

CHRONOS is an elite historical restoration and synthetic media platform. It allows users to:
- **Resurrect Archives:** High-fidelity 8K upscaling and neural colorization.
- **Neural Performance:** Professional face-swapping and actor re-targeting for movies.
- **Compare Slider:** Interactive before/after historical comparison.
- **Restoration Forge:** User-facing AI API for archival media.

---

## 🛠️ Tactical Stack
- **Framework:** [Next.js 14](https://nextjs.org/)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
- **UI:** [Shadcn/UI](https://ui.shadcn.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **API Engine:** [Replicate](https://replicate.com/)

---

## 🚢 Quick Start (Development)

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Deploy to Vercel:**
   ```bash
   vercel deploy
   ```
