# 🏺 Project: CHRONOS – The Sovereign Restoration Engine

**Commander:** ARYA Spike
**Objective:** A high-end entertainment platform for historical restoration and synthetic media showcases.

## 🚢 The Mission
To "Wake the Ancestors" by providing a digital portal that transforms grainy, damaged archival media into hyper-realistic, 4K interactive experiences.

## 🛠️ Tactical Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4.0 + Shadcn/UI
- **Animations:** Framer Motion (for "Time-Travel" effects)
- **Backend:** Supabase (History Vault & Metadata)
- **AI Pipelines:** Replicate API (Topaz-style upscaling, DeepFaceLab-style de-aging)

## 📂 Architecture
- `/src/app/page.tsx` - The Cinematic Hero & Portal
- `/src/components/CompareSlider.tsx` - Interactive Before/After Historical Restoration
- `/src/components/HistoryVault.tsx` - The Gallery of "Woken Ancestors"
- `/src/components/CinemaForge.tsx` - Neural Performance & Movie Face Swap Engine
- `/src/components/RestorationForge.tsx` - The Interactive UI for User-Uploaded Restoration

## 📅 Roadmap (Waves)
1. **Wave 1:** Scaffolding & Hero Design (The Visual Soul)
